This plugin adds the the highslide for image files with Highslide4WP.

== Description ==
This plugin automatically add the class='highslide-image' onclick='return hs.expand(this);' to images linked in a post. You need to install <a href="http://wordpress.org/extend/plugins/highslide4wp/">HighSlide4WP</a> first.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Install <a href="http://wordpress.org/extend/plugins/highslide4wp/">HighSlide4WP</a first
1. Upload `add-highslide.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
